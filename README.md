## About

This repository contains the full marketing website and blog platform for Salt and Light Technologies. It is a production-ready, single-page React application built with a focus on performance, clean design, and owner-operated content management.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | [React 18](https://react.dev) via [Vite](https://vitejs.dev) |
| Styling | [Tailwind CSS v3](https://tailwindcss.com) |
| Routing | [React Router v6](https://reactrouter.com) |
| Backend / Database | [Supabase](https://supabase.com) (Postgres + Storage + Auth) |
| Rich Text Editor | [TipTap](https://tiptap.dev) |
| Contact Forms | [Formspree](https://formspree.io) |
| Hosting | Vercel |
